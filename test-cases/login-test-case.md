Caso de Teste – Login
ID: CT-001
Título: Validar login com credenciais válidas
Pré-condições:

Usuário cadastrado no sistema

Navegador aberto

Passos:

Acessar a página de login

Inserir e-mail válido

Inserir senha válida

Clicar no botão “Entrar”

Resultado Esperado:

Usuário deve ser redirecionado para a página inicial

Nome do usuário deve aparecer no topo da tela
