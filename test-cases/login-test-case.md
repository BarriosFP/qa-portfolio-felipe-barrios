Caso de Teste – Login
ID: CT-001
Título: Validar login com credenciais válidas
Pré-condições:

Usuário cadastrado no sistema

Navegador aberto

Passos:

Acessar a página de login

Inserir e-mail válido

Inserir senha válida

Clicar no botão “Entrar”

Resultado Esperado:

Usuário deve ser redirecionado para a página inicial

Nome do usuário deve aparecer no topo da tela

Caso de Teste – Login com senha incorreta
ID: CT-002
Título: Validar mensagem de erro ao inserir senha incorreta
Pré-condições:

Usuário cadastrado

Passos:

Acessar a página de login

Inserir e-mail válido

Inserir senha incorreta

Clicar em “Entrar”

Resultado Esperado:

Sistema deve exibir mensagem: “Senha incorreta”

Usuário permanece na página de login

Caso de Teste – Campos obrigatórios
ID: CT-003
Título: Validar obrigatoriedade dos campos de login

Passos:

Acessar a página de login

Deixar e-mail e senha vazios

Clicar em “Entrar”

Resultado Esperado:

Sistema deve exibir mensagens de validação nos campos

Login não deve ser permitido
