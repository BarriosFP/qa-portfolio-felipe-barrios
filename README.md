# Portfólio de Qualidade de Software – Felipe Barrios Pinto

Bem-vindo ao meu portfólio de QA!  
Aqui você encontrará meus estudos, casos de teste, bug reports e testes exploratórios desenvolvidos durante minha transição para a área de Qualidade de Software.

---

## 📌 Estrutura do Portfólio

- **test-cases/**  
  Casos de teste manuais escritos com clareza, critérios de aceitação e passos detalhados.

- **bug-reports/**  
  Relatórios de bugs reais encontrados em aplicações, com evidências e passos para reprodução.

- **exploratory-tests/**  
  Testes exploratórios documentados, insights e observações.

- **docs/**  
  Anotações, estudos e materiais de apoio.
